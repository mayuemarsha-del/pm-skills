# docs / 截图放这里

把演示截图 / GIF 放在这个目录，README 里用相对路径引用：

- `demo.gif` — 总览演示（推荐 30-60s 一气呵成跑完一个完整任务）
- `screenshot-prd.png` — PRD skill 触发示意
- `screenshot-acceptance.png` — 验收报告输出示意
- `screenshot-prototype.png` — HTML 原型成品示意

录制建议：
- 用 macOS 内置 `Cmd + Shift + 5` 录屏，转 GIF 用 `ffmpeg -i input.mov output.gif`
- 或者用 [Kap](https://getkap.co/) 直接录成 GIF
- 单个 GIF 控制在 8MB 以下，README 加载会快
